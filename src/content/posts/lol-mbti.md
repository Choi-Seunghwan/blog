## 개발에 앞서

여타 다른 MBTI 테스트 사이트들을 보았을 때 작업 공수가 그리 크지 않아 보여, 3~4일이면 만들 수 있겠다는 생각이 들었습니다....만..., 총 작업 기간은 최초의 계획을 훨씬 상회하는 1.5주 정도 소요되게 되었습니다... 🤣

우선, 개발에 앞서 각 유형 결과에 어떤 내용들을 보여줄 것인지, 질문 내용은 어떻게 작성할 것인지 등 컨텐츠 기획이 필요했고, 그보다 더 앞서 MBTI 라는 게 무엇이며, 어떤 방식으로 검사가 진행되는 것인지, 검사 로직은 무엇인지 등등... 

개발 작업 뿐만 아니라 사전 조사, 사이트 기획 단계에서도 해야 할 일들이 많이 있었습니다.

## 콘텐츠 기획

이제는 모두들 잘 아시겠지만, MBTI는 `I(내향)|E(외향)`, `S(감각)|N(직관)`, `T(사고)|F(감정)`, `J(판단)|P(인식)` 이렇게 각각 네 가지로 대조되는 성향 중 우세한 성향을 조합하여 MBTI 유형을 결정하게 됩니다.

즉, 이는 각 유형에 따른 결과 컨텐츠가 16가지나 필요하단 뜻이었습니다...🥲

정말 컨텐츠 기획을 위해 MBTI 에 대해서 많이 공부했던 것 같습니다. 이왕 만드는 것 완성도 있게 만들고 싶었고, 질문 → 결과까지 이어지는 내용들 역시 ‘이 유형의 유저라면 이렇게 답변할 것이고, 플레이 스타일은 이럴 것이다' 라며 꽤나 고심하면서 문항과 답변을 만들었습니다.

또한 저는 저희 서비스를 홍보하기 위해 ‘리그 오브 레전드' 라는 게임을 배경으로 컨텐츠를 기획하고 있었기 때문에 게임 플레이 스타일이나 공략, 캐릭터 정보 등도 많이 공부하면서 검사 신뢰도가 있는 컨텐츠를 만들고자 노력하였고, 리그 오브 레전드 게임과 MBTI가 서로 잘 조화되도록 노력하였습니다. 꼭 유저가 테스트를 진행하면서 ‘꽤나 그럴싸하네?’ 하고 느끼게끔 하고 싶었습니다. 😄

(그리고 중간 중간 제 기준으로 나름 재밌다고 생각하는 유머 코드들도 넣어 놨습니다 🤭)

콘텐츠 기획을 하는 기간에는 회사 일이 너무 바빠서 여유 시간을 내기가 쉽지만은 않을 때도 있었습니다. 저희 회사는 자율출퇴근제를 시행하고 있는데, 그만큼 자유롭게 근무 시간이 늘어난다는 뜻이었고... 🥲  마침 서비스 리브랜딩을 진행중이던 스프린트여서, 정말 바쁠 때는 일주일 내내 새벽까지 일하고, 다음날 아침 7시까지도 일해서 야근 체크까지 빗겨가는... 수준으로 일 했었습니다.

아무튼! 그럼에도 있는 시간, 없는 시간 쪼개가며 콘텐츠 기획 작업을 진행했고, 노션을 이용하여 질문 문항과 각 유형별 특징, 결과로 보여줄 내용들 등을 정리하였습니다.

## 개발

기술 스택은 다음과 같습니다.

- Vue3 + vite
- AWS Lambda + API Gateway, RDS (Mysql)
- AWS 인프라 세팅, Route 53, S3, CloudFront, 인증서… 등등등…
- Google Analytics, Search Console

FE는 빠른 작업을 위해 능숙한 Vue를 사용하였고, 사용자의 반응을 얻고 싶은 욕심이 생겨 댓글 기능을 추가로 개발하기로 하였습니다. 이를 위해 AWS의 Lambda, API Gateway 등등 간단하게 BE단을 개발하였습니다.

사용자의 답변 통계, 유형별 통계 등 유저 데이터를 수집하기 위해 GA를 붙여 사용자 데이터를 수집하였고, google search console 로 검색 유입이 얼마나 되는지 수집 하였습니다.

## 사이드 프로젝트가 팀에 미치는 영향

팀 단위 사이드 프로젝트 뿐만 아니라, 개발자 개인의 사이드 프로젝트가 팀에 미치는 좋은 영향 역시 있었던 것 같습니다.

저와 비슷한 연차의 동료 분들과 사이드 프로젝트에 관해 이야기 하면서 공동의 향상심을 키우게 되기도 하였고, 공부 의욕도 함께 많이 생겼던 것 같습니다.

팀 단위에서도 우리 서비스를 홍보하는 사이트가 생겼다는 것에 대한 흥미로운 감정들이 생겼던 것 같습니다.

또한 경영진의 시각에서도 활기 있는 팀의 모습을 보여주게 된 것 같아, 사내 여러 이해관계에 있어서 팀의 입지가 조금..? 좋아지지 않았을까… 하는 생각도 듭니다

하지만 역시, 결과적으로 제일 중요한 것은, 저와 같은 주니어 개발자들의 성장 의욕을 함께 높였다는 것에 있는 것 같습니다.
