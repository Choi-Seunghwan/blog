# chuz.dev

개인 기술 블로그. 백엔드 개발, 인프라, 모니터링 등 실무 경험과 기술 지식을 정리합니다.

> https://chuz.dev

## Tech Stack

- **Framework**: [Astro](https://astro.build/) v5
- **Theme**: [Astro Terminal Theme](https://github.com/dennisklappe/astro-theme-terminal) (Hugo Terminal Theme 포팅)


## 프로젝트 구조

```
src/
├── content/posts/    # 블로그 글 (Markdown)
├── pages/            # 페이지 (About, Tags 등)
├── layouts/          # 레이아웃 (Base, Post)
├── components/       # 컴포넌트 (Header, Footer, PostCard 등)
└── styles/           # CSS (terminal.css, code.css 등)
public/
└── images/posts/     # 포스트 이미지
```
